Програмний продукт: Система дистанційного навчання EduFlow

Теми і цілі

Основною темою роботи є створення сучасної, високопродуктивної системи для організації дистанційного навчання. Основні цілі:

  • Забезпечення безперебійної взаємодії між ічителем та учнями.
  • Автоматизація перевірки тестових завдань.
  • Створення зручного інтерфейсу для управління навчальними матеріалами.

Опис застосування

EduFlow дозволяє створювати віртуальні класи, завантажувати лекції у форматі PDF, призначати завдання з дедлайнами та проводити тестування в режимі реального часу.

Мови програмування

TypeScript (Frontend & Backend)

Засоби розробки

VS Code, Bun Runtime, Vite, Git, MongoDB Compass

Технічні вимоги

  • ОС: Windows 10+, Linux або macOS.
  • Node.js версії 20 або вище.
  • Браузер з підтримкою ES6+.

Алгоритм роботи системи

1

Авторизація та визначення ролі (Студент/Викладач)

↓
2

Отримання даних через API (Hono/Bun)

↓
3

Рендеринг інтерфейсу (React/Next.js)

↓
4

Взаємодія в реальному часі (WebSockets)

Апробація: Продукт протестовано локально та розгорнуто для тестової групи користувачів (5 осіб). Багів не виявлено.

Участь у виробничій та науковій роботі

Під час практики було проведено дослідження методів оптимізації передачі даних у WebSockets для великої кількості одночасних підключень.

Результати наукових досліджень

Основні результати, отримані з кваліфікаційної роботи, впроваджено в модуль чату EduFlow, що дозволило зменшити затримку повідомлень на 15%.
